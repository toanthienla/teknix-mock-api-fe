## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone repository:

```bash
git clone https://github.com/toanthienla/teknix-mock-api-fe.git
cd teknix-mock-api-fe/mock-api
```

2. Install dependencies:

```bash
npm install
```

3. Run development server:

```bash
npm run dev
```
